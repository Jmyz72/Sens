# Screenshots

Drop product screenshots here and reference them from the root [`README.md`](../../README.md).

## Suggested shots

| Filename | What to capture |
| -------- | --------------- |
| `dashboard.png` | The dashboard with monthly income/expense, net cashflow, and spending breakdown |
| `accounts.png` | The Accounts screen grouped by type, showing balances and provider logos |
| `transactions.png` | The Transactions list with the different kinds (income/expense/transfer/adjustment) |
| `add-account.png` | The two-step New Account modal (provider picker → type/subtype) |

## Tips

- Capture in **both dark and light** themes if you want to show off theming.
- Use realistic-but-fake MYR figures — never real personal financial data.
- Keep a consistent window size for a tidy README table.
- PNG is preferred for crisp UI; keep files reasonably sized.
